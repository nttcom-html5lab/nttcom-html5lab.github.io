2014年1月18日に開催される「Web Musicハッカソン #2」に、WebRTCプラットフォーム「SkyWay」を提供し、チュータとしてもお手伝いすることになりました。

Web Music ハッカソンとは、ブラウザで音声処理・信号処理を行うWeb Audio API、外部 MIDI デバイスと連携をするWeb MIDI API など、ブラウザの持つ音楽に関わるAPIを使って、一日でアプリケーションを制作しよう、というイベントです。 WebRTCのP2P通信と組み合わせると面白い作品を作り上げられるのではないかと考え、HTML5ラボが提供する「SkyWay」を提供し、チュータとしてもお手伝いすることになりました。

イベントの詳細等は、Google japan Developer Relation Blogをご覧ください。
